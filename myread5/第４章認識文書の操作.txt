MYREAD5　第４章　認識文書の操作

4-1. カーソルの移動

4-1-1. カーソルを移動する

■基本操作
　左の文字へ移動         ←
　右の文字へ移動         →
　上の行へ移動           ↑
　下の行へ移動           ↓

　行の先頭が行頭で、最後の文字が行末です。
　行頭へ移動             Home
　行末へ移動             End

　文章の先頭から、改行文字で行が改められるまでのブロックが段落です。
　前の段落へ移動         Ctrl + ↑
　次の段落へ移動         Ctrl + ↓

　文書の先頭が文頭で、最後が文末です。
　文頭へ移動             Ctrl + Home
　文末へ移動             Ctrl + End
　音声では『トップポジション』『ラストポジション』とガイドします

■表内の移動
　表を使っている原稿では、認識文書にも その表イメージが取り込まれます。
　表は、縦横の並びを マス目状に罫線で区切って表し、縦の並びを列、横の並びを行と呼びます。罫線で区切られた１つ１つのマス目を セルと呼び、このセルの中に文章が書き込まれています。
　↑ ↓ ← → を押して、表内をなぞるようにカーソル移動します。
　この場合、罫線の上にもカーソルは移動します。

　次のセル、前のセルというように、セルからセルへ移動できます。
　次のセルへ移動         Ctrl + →
　前のセルへ移動         Ctrl + ←

■複数原稿の文書移動
　複数の原稿を連続読み取りしたときや、認識文書を追加しながら読み取りしたとき
　各原稿の先頭位置へ移動できます。
　現在の原稿の先頭へ移動 Ctrl + Shift + Alt + Home
　前の原稿へ移動         PageUp
　次の原稿へ移動         PageDown

■ページの移動
　認識文書は、文書の行数が多くなるとページに分割されます。
　具体的には、A4の用紙で計算したときの１ページ最大行数で分割されます。
　これは、読み取った原稿の用紙サイズとは関係なく、認識文書をページ管理して
　わかりやすくするためです。
　前のページへ移動       Ctrl + PageUp
　次のページへ移動       Ctrl + PageDown

4-1-2. 現在のカーソル位置を音声で知る

カーソル位置の読み上げ形式には ２つあります。
1.原稿単位のカーソル位置
　原稿番号 行 列の順で『原稿 1の19の8』のように読み上げます。
　これは、認識文書を各原稿 先頭からの位置で知ることができます。
2.ページ単位のカーソル位置
　文書のページ番号 行 列の順で『A4 1の3の5』のように読み上げます。
　これは、認識文書を A4用紙のページ位置で知ることができます。

　原稿単位のカーソル位置 読み上げ    F9
　ページ単位のカーソル位置 読み上げ  Ctrl + Shift + F9

4-1-3. カーソル移動時の文字の読み方を変更する

カーソル移動したときの文字の読み方には、
　簡易読み      簡単な 易しい説明の読みです
　音訓読み      六点漢字の 音読みと訓読み で表現した読みです
　音読み        音読みです
　詳細音訓読み  簡易読みよりも詳しい読みです
　詳細区分読み  詳細音訓読みに､漢字の区分を加えた読みです
　識別読み      全角 半角や、文字の種類も加えた詳しい読みです
　ビープ        簡単な電子音を出力します
と、７つの読み方があります。
F8で、これらの読み方を順番に切り替えることができます。

4-1-4. カーソル移動の点字コマンド

　↑                      2 5の点 + Space
　↓                      3 6の点 + Space
　→                      2の点 + Space
　行頭に移動              せ + 無変換（ 行の先頭の せ ）
　行末に移動              ら + 無変換（ 行のラストの ら ）
　前の段落に移動          2 5の点 + 無変換
　次の段落に移動          3 6の点 + 無変換
　文頭に移動              と + Space（ 文書のトップの と ）
　文末に移動              ら + Space（ 文書のラストの ら ）
　前の原稿に移動          は + Space（ バックの は ）
　次の原稿に移動          ね + Space（ ネクストの ね ）
　前のページに移動        は + 無変換（ バックの は ）
　次のページ移動          ね + 無変換（ ネクストの ね ）
　カーソル位置の読み上げ  か + Space（ カーソル位置の か ）

4-2. 認識文書の読み上げ

4-2-1. 文書を読み上げる

■全文読み
　全文読みのコマンドには、次の２つがあります。
　Alt + F5で  文頭から読み上げを開始します。
　Alt + F10で 現在のカーソル位置から読み上げを開始します。
読み上げ中は、カーソルが、読み上げている場所を追いかけながら進んでいきます。
文書の最後まで読み上げたら『おわり』と音声ガイドして読み上げを終了します。
読み上げを中断したいときは、Escapeを押します。『パス』と音声ガイドし、読み上げを終了します。このとき、カーソル位置は、最後に読み上げていた文章の近くまで進んできています。ですから、ここだぁ と思った内容が読み上げられたら、Escapeで読み上げを停止して、その場所までカーソル移動するといいですね。

■全文読み上げ中の操作
　全文読みの最中に、内容を聞き逃してしまったり、もう一度 注意深く聞いてみたいときは、読み上げ位置を戻すことができます。逆に、今聞いている内容をスキップさせたいときは、読み上げ位置を進めることができます。
　以下に、全文読み上げ中の操作コマンドを紹介します。

　段落単位で読み上げ位置を移動します。
　段落の先頭に戻る       ←
　次の段落先頭に進む     ↓
　前の段落先頭に戻る     ↑

　空白や句読点で区切られた文節へ、読み上げ位置を進めます。
　次の文節に進む         →

　行単位で読み上げ位置を移動します。
　次の行へ進む           Enter
　前の行へ戻る           BackSpace

　原稿単位で読み上げ位置を移動します。
　原稿の先頭へ戻る       Home
　次の原稿へ進む         PageDown
　前の原稿へ進む         PageUp

　また、全文読み上げ中に 音声スピード・音声ボリュームを切り替えできます。
　音声スピード切替       F7
　音声ボリューム切替     F11

■１行読み
　カーソル行の読み上げ   Alt + F8

　行頭からカーソル位置の手前までの読み上げ
　　PC-Talker            Ctrl + Alt + H
　　VDM                  Ctrl + Alt + J

　カーソル位置から行末までの読み上げ
　　PC-Talker            Ctrl + Alt + K
　　VDM                  Ctrl + Alt + L

■段落読み
　Alt + F9で カーソルのある段落の先頭から、その段落の最後までを読み上げます。

■表の読み上げ
　全文読み上げ中は、表をセル単位で 順番に読み上げていきます。

　セル内の文章を始めから終わりまで読み上げ確認する便利なコマンドがあります。
　表のセル内読み         Alt + F7

■カーソル上の文字情報を読み上げる、いろんなコマンドを紹介します。
　カーソル文字の再読み
　カーソル上の文字を再度読み上げます。
　　PC-Talker            Ctrl + Alt + R
　　VDM                  Ctrl + Alt + G

　カーソル文字の点字符号読み
　KTOSご使用の場合、六点漢字の符号で読み上げます。
　KTOS8ご使用場合、漢点字の符号で読み上げます。
　　PC-Talker            Ctrl + Alt + F
　　VDM                  Ctrl + Alt + X

　カーソル文字のJISコード読み
　シフトJISコードに続けて、JISコードの順番で読み上げます。
　　PC-Talker            Ctrl + Alt + I
　　VDM                  Ctrl + Alt + N

4-2-2. 読み上げの点字コマンド

　カーソルから全文読み          お + Space （ オールの お ）
　文頭から全文読み              お + 無変換 （ オールの お ）
　１行読み                      わ + Space （ ワンラインの わ ）
　段落読み                      る + Space （ 以前は、ループ読みと呼んでいた ）
　行頭からカーソル手前読み      れ下がり + 無変換  ※PC-Talkerのみ
　カーソルから行末読み          ら下がり + 無変換  ※PC-Talkerのみ

以下は、六点キー周辺に配置されているキーで実行します。
　カーソル文字の再読み          1の点 右隣   G
　カーソル文字の点字符号読み    3の点 右下   X
　カーソル文字のJISコード読み   3の点 左下   Z

4-3. 認識文書の編集

マイリードの初期設定では、認識文書に対し文字の追加や修正・削除といった編集ができないようになっています。これは、認識オプションの設定で編集できるように変更できます。編集オプションの設定についてはあとで説明しています。
※文書のコピー、検索の編集機能は、設定に関係なく使用できます。

4-3-1. 文字の挿入・削除

文字の入力や、文字の削除といった 基本的な編集操作は、通常のエディタや ワープロと同じです。PC-Talker付属のエディタ MYEDITのマニュアルでは、文字入力の基礎から やさしく解説していますので、そちらもぜひご覧ください。
ここでは、認識文書で使用できる文字の挿入・削除機能を紹介します。

■挿入と上書き
　文字を入力する方法には、挿入モードと上書モードがあります。
　挿入モード      入力した文字は、カーソル位置に挿入されます。
　上書モード      カーソル上の文字が入力した文字で上書きされます。
　また、上書モードの文字入力を助ける音声機能に 先読みエコーがあります。これは、
　入力した文字の読みに続けて、次に上書きされるカーソル上の文字を読む機能です。
　どこまで上書きしたのか、音声で確かめながら入力できますね。

　挿入・上書の切替            Insert
　先読みエコーの切替          Ctrl + Shift + E

■改行・改ページの挿入
　改行文字や 改ページ文字は、上書モードのときもカーソル位置に挿入されます。
　改行の挿入                  Enter
　改ページの挿入              Ctrl + Shift + Enter

■文字の削除
　文字単位の削除には、Delete または BackSpaceを使います。
　カーソル上の文字削除        Delete
　カーソル手前の文字削除      BackSpace

　カーソル上の文字から 右側の文字列すべてを削除できます。
　カーソル以降の右削除        Ctrl + Delete

■全文の削除
　認識文書をすべて削除できます。
　全文削除                    Ctrl + Shift + A

4-3-2. コピー・切り取り・貼り付け

コピー、切り取り、貼り付けのコマンドを使って、文書の一部を別の場所へ、複写したり移動したりできます。その場合、文書の一部は範囲選択の操作をおこなって指定します。
範囲選択の操作には、Shift + カーソルキーによるWindows標準のやり方がありますが、ここでは、初心者のかたや、視覚障害者のかたにも扱いやすい独自の範囲選択の操作を最初に紹介します。

■範囲選択の操作
①範囲選択したい最初の文字へカーソル移動し、Alt + Enterを押します。
　これが、範囲選択の開始コマンドです。
　このときのカーソル文字が、範囲の先頭文字として確定されます。
②『範囲指定 文字 どこまで』のガイドに従い、範囲の終了文字へカーソル移動します。
　ここで Spaceを押して、範囲選択の単位を変更できます。
　単位には、次の３つがあります。
　　文字単位        範囲の先頭文字から終了文字までを範囲選択します。
　　行単位          範囲の先頭行から終了行までを範囲選択します。
　　頁単位          範囲の先頭頁から終了頁までを範囲選択します。
③最後に Enterを押して、範囲選択を完了します。

　Windows標準の操作と違い、範囲選択が完了するまで文字キーの入力による更新を禁止していますので、安心して範囲選択の操作ができます。

　範囲選択の中止
　範囲選択の操作を中止するには、Escapeを押します。

　選択した範囲の解除
　範囲選択の完了後に、選択した範囲を解除するには、Escapeを押します。
　このとき『範囲クリア』と音声ガイドします。

　選択した範囲の文書を読み上げる
　Alt + F10を押して 全文読みを実行します。
　選択されている範囲内の文書を読み上げます。

　文書全体を範囲選択したいとき
　Ctrl + Aを押して 簡単に文書全体を範囲選択できます。

■複写と移動
①複写 または 移動したい文書データを範囲選択します。
②複写する場合はコピーを、移動する場合は切り取りを実行します。
　コピー       Ctrl + C または Ctrl + Insert
　切り取り     Ctrl + X または Shift + Delete
③複写 または 移動したい位置へカーソルを移動して、貼り付けを実行します。
　貼り付け     Ctrl + V または Shift + Insert

　コピーしたり切り取りした文書データは、別のアプリケーションに貼り付けることができます。ご利用になりたいアプリケーション上で貼り付けを実行してください。
MYEDITや MYWORDでは、同じコマンドで貼り付けできますが、他のアプリケーション上での貼り付け方法は、そのアプリケーションのマニュアルをご覧ください。
　また、表データをそのままの形で貼り付けできるアプリケーションは、MYWORD5だけです。他のアプリケーションでは、表セル内の文字データだけが貼り付けられます。

4-3-3. 検索・置換

■検索
　認識文書の中から指定した文字列を検索します。
①検索を始めたい位置へ、カーソル移動します。
②Ctrl + Fを押して、検索の実行画面を開きます。ここで 検索条件を指定します。
　検索条件には、以下の項目があります。
　各項目は、Tabを押して移動します。
　●【検索する文字列】
　検索したい文字列を入力します。最近検索した文字列は、↓↑を押して選択できます。
　↓↑で 項目が移動するときは、Ctrl + Spaceを押して選択できます。

　●【履歴削除】
　最近検索した文字列の履歴を削除したいとき、ここで Enterを押します。
　履歴が削除されたあと、【検索する文字列】の項目へ戻ります。

　●【完全一致で検索する】
　この項目は、Spaceを押して チェックを切り替えます。
　『チェック』       検索する文字列と、完全に一致する文字列を探します。
　『チェックなし』   検索する文字列と、文字種の違いを同一視して探します。

③最後に［実行］まで移動して、Enterを押すと検索を実行します。
　文字列が見つかった場合、そのカーソル位置を音声ガイドします。
　文字列が見つからなかった場合、『検索文字が見つかりません』と音声ガイドします。

　検索を実行したあと、同じ検索条件で 次を検索したり、前を検索したりできます。
　次を検索     Alt + ↓
　前を検索     Alt + ↑

■置換
　認識文書の中から指定した文字列を検索し、別の文字列に置き換えます。
①置換を始めたい位置へ、カーソル移動します。
②Ctrl + Hを押して 置換の実行画面を開きます。
　ここでは、上で説明した検索と同じ要領で、置換条件を指定します。
　●【検索する文字列】   ここへ、置換前の古い文字列を入力します。
　●【置換する文字列】   ここへ、置換後の新しい文字列を入力します。
　あとの項目は、検索で説明した内容と同じです。

③次に置換を実行してからの操作を説明します。
　文字列が見つかった場合、置換確認の問い合わせがあります。
　１つずつ確認しながら置換する場合は［置換］を、置換しない場合は［次を検索］を
　すべて一括して置換する場合は［すべて置換］を← →で 選択して Enterを押します。
　置換を中止したいときは［キャンセル］を選択するか、Escapeを押します。
　文字列が見つからなかった場合、『検索文字が見つかりません』と音声ガイドします。

4-3-4. 元に戻す・やり直し

　元に戻す     Ctrl + Z
　直前におこなった編集操作が取り消され、編集前の状態に戻ります。

　やり直し     Ctrl + Y
　取り消された編集操作を再実行します。

4-3-5. 編集の点字コマンド

　切り取り             ぬ + Space  （ 抜き取るの ぬ ）
　コピー               ほ + 無変換 （ 一時保存の ほ ）
　貼り付け             つ + 無変換 （ 貼り付けの つ ）
　元に戻す             六点キー周辺に配置　P
　やり直し             り + 無変換 （ リドゥの り ）
　検索                 も + Space  （ 文字検索の も ）
　次を検索             4 6の点 + Space
　前を検索             4 5の点 + Space
　置換                 ち + Space  （ 置換の ち ）

4-4. 認識文書と認識画像の連動について

認識文書のカーソルが移動すると、それに対応する認識画像上の文字位置が反転して表示されます。これは、弱視者や晴眼者のかたにとって、カーソル上の文字が、認識画像のどの部分に当たるのかを知るのに便利であり、簡単に認識文書と認識画像の文字を見比べながら編集できます。
認識画像が見えにくい場合は、画像の拡大倍率を変更して見やすくできます。これについては「第５章　拡大読書」をご覧ください。

もう１つ、認識画像について説明します。
Ctrl + Tabを押すと、読み取りの選択メニュー、認識画像、認識文書の順番で 操作画面を移動することができます。認識画像の操作に移ってみましょう。
認識画像では、認識領域の注目位置を移動できます。このとき、認識文書も連動して、移動した認識領域の先頭文字へカーソル移動します。
認識領域とは、文章や図の書かれている部分を四角い枠で表示した領域のことです。
認識領域の移動コマンドは、次の通りです。
　次の領域へ移動         Space
　前の領域へ移動         Shift + Space
　先頭領域へ移動         Home
　最終領域へ移動         End

4-5. 認識文書の保存

認識文書を ファイル保存できます。
保存したファイルは、別のアプリケーションで読み込み 利用できます。

認識文書を保存するには、Ctrl + S または Ctrl + Shift + F2を押します。
点字コマンドは、せ + Space（ セーブのせ ）です。
認識文書の保存画面が開き、ここで ファイル名などを設定してから保存します。
　各設定項目は、Tabを押して移動します。
　保存を中止したいときは、Escapeを押します。

次に、各設定項目の操作について説明します。
●【ファイル名】
　ここに保存ファイルの名前を入力します。
　その名前で保存する場合は Enterを押します。
　ファイルの拡張子によって、保存するファイル形式を指定できます。
　　txt        テキスト形式で保存します。
　　htm        HTML形式で保存します。
　　html       HTML形式で保存します。
　　MWE        MYWORD5の文書形式で保存します。
　拡張子を省略した場合は、テキスト形式で保存します。

●【一覧】
　ここには、既存のファイル名や フォルダ名が一覧されています。
　↓↑で ファイル名や フォルダ名を選択します。
　ファイル名を選択して Enterを押すと、そのファイルに上書保存できます。
　フォルダ名を選択して Enterを押すと、そのフォルダ内の一覧に変更できます。
　一覧先のフォルダやドライブ番号が変更されると、保存先もその場所に変更されます。
　【現在のドライブ】を変更して 一覧先のドライブ番号を変更することもできます。
　【ファイルの種類】を変更して 一覧するファイルの種類を変更することもできます。

●【現在のドライブ】
　← →で ドライブ番号を設定します。

●【ファイルの種類】
　← →で ファイルの種類を 次の中から設定します。
　〈テキスト(*.txt)〉
　〈HTML文書(*.htm, *html)〉
　〈MYWORD5文書(*.MWE)〉
　〈全てのファイル(*.*)〉

4-6. 認識文書の印刷

プリンタを使って、認識文書を印刷します。
プリンタには、A4用紙を縦向きにセットして印刷してください。
認識文書を印刷するには、Ctrl + P または Ctrl + Shift + F6を押します。
点字コマンドは、ふ + 無変換（ プリンタのふ ）です。
印刷の実行画面が開き、ここで 印刷範囲や 印刷部数を設定してから実行します。
　各設定項目は、Tabを押して移動します。
　設定内容を選択するには、← → を押してください。
　最後に［実行］まで移動して、Enterを押すと印刷を実行します。
　印刷を中止したいときは、Escapeを押します。

それでは、各設定項目について説明します。
●【印刷範囲】
　印刷する範囲を 次の中から設定します。
〈文書全体〉        すべての文書を印刷します。
〈現在のページ〉    カーソルのあるページだけを印刷します。
〈指定した範囲〉    範囲選択されている文書だけを印刷します。

●【印刷部数】
　印刷する部数を 数値で入力します。

4-7. 認識オプションを設定する

認識オプションでは、認識文書の書式や、自動ファイル保存について設定できます。

認識オプションを設定するには、プルダウンメニュー［環境-認識オプション］を実行します。または、Shift + F4を押して実行します。
認識オプションの設定画面が開きます。
　各項目は、Tabを押して移動します。
　チェック項目は、Spaceを押して チェック状態を切り替えます。
　選択項目は、← →を押して変更します。
　最後に［設定］まで移動して、Enterを押すと設定を完了します。
　設定を中止したいときは、Escapeを押します。

それでは、各設定項目について説明します。
●【認識文書に 原稿番号を入れる】
　『チェック』       認識文書の先頭行に「*原稿番号」の形式で挿入します。
　『チェックなし』   原稿番号を挿入しません。
　複数枚の原稿を読み取った場合、各原稿の文書の始まりを分かりやすくするために
　原稿番号を挿入しておくと便利です。

●【全角の英数記号を半角に変換する】
　『チェック』       認識文書の 英字、数字、記号を半角文字で作成します。
　『チェックなし』   全角文字で作成します。
　一般的に、英単語や 金額などの数値は、半角文字のほうが 見やすいです。

●【認識文書の編集を可能にする】
　『チェック』       文字の追加や修正・削除といった編集ができるようになります。
　『チェックなし』   コピー、検索以外の編集機能は使えません。

●【認識文書の自動ファイル保存】
　読み取りの実行後、認識文書を自動的にファイル保存するか 次の中から設定します。
　〈保存しない〉     ファイル保存しません。
　〈上書きで保存〉   既存のファイルに上書き保存します。
　〈追加で保存〉     既存のファイルに追加して保存します。
　保存するファイル名は、次の設定項目で指定できます。

●【自動保存するファイル】
　認識文書を自動保存するファイル名を入力します。
　ファイル名は、保存先のフォルダ名も含めて入力してください。
　保存先のフォルダを変更したい場合、次の［フォルダの参照］を使うと便利です。

●［フォルダの参照］
　ここで、Enterを押すと フォルダの参照画面が開きます。
　保存先に指定したいフォルダを一覧の中から決定してください。一覧の基本操作は
　↑↓を押して「フォルダ名」を選択し Enterで そのフォルダの中へ移動できます。
　Homeを押して「上のフォルダへ」を選択し Enterで １つ上のフォルダに戻ります。
　Endを押して「現在のフォルダで決定」を選択し Enterで フォルダを決定します。

※認識オプションの他、マイリードの環境設定値を出荷時の値に戻したいときは、
　プルダウンメニュー［ヘルプ-Myread5の初期化］を実行してください。
