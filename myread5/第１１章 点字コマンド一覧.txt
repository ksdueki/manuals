マイリードの点字コマンドを一覧します。

MYREAD5　第１１章　点字コマンド一覧

ここに記載されていない機能のコマンドは、「第１０章　フルキーコマンド一覧」をご覧ください。

※注意
KTOSの漢点字入力を使用しているかたは、「点字符号+無変換」の部分を「点字符号+変換」に置き換えて使用してください。


●全般
  マイリードの終了                 め + Space
  マニュアルを読む                 め + 無変換
  プルダウンメニューの操作に移る   6の点 + 無変換
  Enterの代替                      あ + Space
  Escapeの代替                     6の点 + Space


●拡大読書画面の操作
  画像の右90度回転                 か + 無変換
  画像の表示色 反転                い + 無変換
  拡大読書画面の終了               6の点 + Space または め + Space
  》以下は、認識画像の拡大読書中で有効
      先頭領域へ 表示位置を移動    と + Space
      最終領域へ 表示位置を移動    ら + Space


●データの保存と印刷
  認識文書の保存                   せ + Space
  認識文書の印刷                   ふ + 無変換


▼【認識文書】画面の操作
●【認識文書】…… カーソル移動
  左の文字へ                       4の点の上
  右の文字へ                       2の点 + Space
                                   5の点の上
  上の行へ                         2 5の点 + Space
                                   6の点の上
  下の行へ                         3 6の点 + Space
                                   6の点の右下
  行頭へ                           せ + 無変換
  行末へ                           ら + 無変換
  前の段落へ                       2 5の点 + 無変換
  次の段落へ                       3 6の点 + 無変換
  文頭へ                           と + Space
  文末へ                           ら + Space
  現在の原稿の先頭へ移動           ほ + Space
  前の原稿へ移動                   は + Space
  次の原稿へ移動                   ね + Space
  前のページへ移動                 は + 無変換
  次のページへ移動                 ね + 無変換
●【認識文書】…… カーソル文字の読み上げ
  カーソル位置の読み上げ           か + Space       ※『原稿番号 行 列』
  カーソル移動時の読み方切替       4の点の左下
  フォネティック読みのオンオフ     5の点の右下
  点字符号読み                     2の点の左下
  JISコード読み                    3の点の左下
●【認識文書】…… なめらか読みの読み方切替
  句点・括弧の読み方切替           1の点の右下右
  数字の読み方切替                 1の点の右下
  英語の読み方切替                （Windows98/ME）     4の点の左上左
                                  （WindowsXP/Vista/7）6の点の右下右
●【認識文書】…… 文書の読み上げ
  １行読み                         わ + Space
  行頭からカーソル手前読み         れ下がり + 無変換
  カーソルから行末読み             ら下がり + 無変換
  段落読み                         る + Space
  表セル内読み                     わ + 無変換
  文頭から全文読み                 お + 無変換
  カーソルから全文読み             お + Space
  読み上げの中断                   6の点 + Space
●【認識文書】…… 日本語変換中の読み上げ
  反転表示内の読み上げ             れ下がり + 無変換
  反転表示以降の読み上げ           ら下がり + 無変換
  日本語変換の読み方切替           3の点の上
●【認識文書】…… その他の読み上げ
  入力状態の読み上げ               に + 無変換
  ページ設定の読み上げ             へ + 無変換
  クリップボードの内容読み上げ     く + 無変換
●【認識文書】…… 文字の挿入・削除
  挿入・上書の切替                 い + Space
  先読みエコーの切替               え下がり + Space
  改行の挿入                       あ + Space
  改ページの挿入                   を + Space
  カーソル上の文字削除             5の点 + Space
  カーソル手前の文字削除           4の点 + Space
  カーソル以降の右削除             れ + Space
  全文削除                         く + Space
●【認識文書】…… 編集
  元に戻す                         6の点の右上
  やり直し                         り + 無変換
  切り取り                         ぬ + Space
  コピー                           ほ + 無変換
  貼り付け                         つ + 無変換
  検索                             も + Space
  次を検索                         4 6の点 + Space
  前を検索                         4 5の点 + Space
  置換                             ち + Space


▼【画像】画面の操作（取込画像と認識画像）
●【画像】…… 表示
  画像の表示色 反転                い + 無変換    ※認識画像に有効
●【画像】…… 取込画像の操作
  取込画像の右90度回転             か + 無変換
  取込画像のコピー                 ほ + 無変換    ※画像画面上の操作
●【画像】…… 認識画像の操作
  ※以下は画像画面上の操作
  領域の削除                       5の点 + Space


●その他
  日時読み上げ                     と + 無変換
