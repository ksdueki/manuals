ここでのメディアとは、音声や音楽、または、動画が収録されたファイルのことです。
サウンドやミュージック、オーディオやビデオなどと細かく分類されることもありますが、それらも含まれた総称になります。
また、オーディオブックCDについても、話者が読み上げた書籍などをメディアファイルに録音、そのファイルをCDに格納しているものですから、メディアファイルと同じ操作方法で再生できます。
ここでは、これらメディアファイルの再生中の操作方法を説明します。

次のコマンドで、ディスクに格納したメディアファイルやCDに納められたメディアファイルを再生し、音楽や動画をお楽しみください。

メディアの再生中、一時停止中には、次のコマンドを使用できます。

　・再生の一時停止と再開　　　　　　Space または Enter
　・再生の早送り　　　　　　　　　　→
　・再生の巻き戻し　　　　　　　　　←
　・先頭まで巻き戻して再生　　　　　Home
　・指定時間からの再生　　　　　　　Ctrl + J
　　※点字コマンドでは　　　　　　　3 4 5 6の点 + Space
　　※再生を開始する時間【分】【秒】を数値で入力し、Enterを押します。
　　　入力した時間の位置から再生が始まります。
　・ミュート　　　　　　　　　　　　テンキーの数字 0
　　※ミュートとは、音を消したまま再生させることです
　・音量を上げる　　　　　　　　　　Ctrl + ↑
　・音量を下げる　　　　　　　　　　Ctrl + ↓
　　※この音量は、再生音の大きさであり、PC-Talkerの音量には影響しません
　・再生スピードを上げる　　　　　　Ctrl + →
　・再生スピードを下げる　　　　　　Ctrl + ←
　・再生スピードを標準に戻す　　　　Ctrl + テンキーの数字 0
　　※録音データの形式により、変更できない場合もあります。
　・右スピーカーのバランスを上げる　Ctrl + Shift + →
　・左スピーカーのバランスを上げる　Ctrl + Shift + ←
　　※これは、ステレオスピーカーの音量バランスを調整する機能です
　・画像の大きさ切り替え　　　　　　　　テンキーの数字 1
　　※画像をウィンドウサイズに合わせるか、通常サイズにするかを切り替えます
　・メディア情報読み上げ　　　　　　Alt + Ctrl + G
　　※次のメディア情報を読み上げます。
　　　1.メディアタイトル
　　　2.メディアタイプ　『ビデオ』『オーディオ』など
　　　3.メディア全体の再生時間
　　　4.メディアの著作権
　　　5.現在の状態　　　『再生中』か『一時停止』
　・現在の再生位置の読み上げ　　　　Alt + Ctrl + F9
　　※現在の再生位置までの時間と、全体の再生時間を読み上げます
　・拡張メニュー　　　　　　　　　　Alt + K
　　※拡張メニューを開き、早送りや巻き戻しをこちらからも行えます。
　　　また、「リピート再生する」で Enterを押すと、リピート再生になります。

　・再生の終了　　　　　　　　　　　EscapeまたはBackSpace
