マイブックで取り扱えるデイジーは次の通りです。
　1.CDで図書が提供される デイジーCD
　2.サピエ図書館などネット配信されているデイジー図書
　3.マイブックの本棚のようにディスクに保存したデイジー図書
このように複数の媒体が取り扱えますが、どれも同じ操作方法でお楽しみいただけます。
ただ、注意していただきたいことは、
デイジーには、音声デイジーとマルチメディアデイジー、それにテキストデイジーの3種類の形式があります。
これらは、再生方式が次のように異なります。
【音声デイジーとマルチメディアデイジー】は、話者の録音データを再生します。
【テキストデイジー】は、テキストデータを音声で読み上げます。

再生方式により、コマンドとショートカットキーが異なりますので、２つに分けて説明します。

1.【音声デイジーとマルチメディアデイジー】
　再生中、一時停止中に、次のコマンドを使用できます。
　・再生の一時停止と再開　　　　　　Space
　・次のフレーズ　　　　　　　　　　→
　・前のフレーズ　　　　　　　　　　←
　・次の見出し　　　　　　　　　　　↓
　・前の見出し　　　　　　　　　　　↑
　・見出し移動のレベル切り替え　　　Ctrl+フルキーの数字キー　※以降、補足１を参照
　・次のページ　　　　　　　　　　　PageDown
　・前のページ　　　　　　　　　　　PageUp
　・ページジャンプ　　　　　　　　　Ctrl + J
　・次のしおり　　　　　　　　　　　M
　・前のしおり　　　　　　　　　　　Shift + M

　・ミュート　　　　　　　　　　　　テンキーの数字 0
　　※ミュートとは、音を消したまま再生させることです。
　・音量を上げる　　　　　　　　　　Ctrl + ↑
　・音量を下げる　　　　　　　　　　Ctrl + ↓
　　※この音量は、再生音の大きさであり、PC-Talkerの音量には影響しません。
　・再生スピードを上げる　　　　　　Ctrl + →
　・再生スピードを下げる　　　　　　Ctrl + ←
　・再生スピードを標準に戻す　　　　Ctrl + テンキーの数字 0
　　※録音データの形式により、変更できない場合もあります。
　・右スピーカーのバランスを上げる　Ctrl + Shift + →
　・左スピーカーのバランスを上げる　Ctrl + Shift + ←
　　※これは、ステレオスピーカーの音量バランスを調整する機能です
　・カーソル位置 読み上げ 　　　　　F9
　　※現在のページ番号と 残り再生時間を読み上げます。

　・再生の終了　　　　　　　　　　　EscapeまたはBackSpace

2.【テキストデイジー】
　読み上げ中、読み上げ停止中、共に次のコマンドを使用できます。
　・次のフレーズ　　　　　　　　　　→
　・前のフレーズ　　　　　　　　　　←
　・次の段落　　　　　　　　　　　　↓
　・前の段落　　　　　　　　　　　　↑
　・次のページ　　　　　　　　　　　PageDown
　・前のページ　　　　　　　　　　　PageUp
　・次の見出し　　　　　　　　　　　Ctrl + PageDown
　・前の見出し　　　　　　　　　　　Ctrl + PageUp
　・次のしおり　　　　　　　　　　　M
　・前のしおり　　　　　　　　　　　Shift + M
　・読み上げの終了　　　　　　　　　EscapeまたはBackSpace

　・音声スピード切り替え　　　　　　F7
　・音声ボリューム切り替え　　　　　F11
　
　・カーソル位置 読み上げ 　　　　　F9
　　※合計のページ数と 現在のページを読み上げます。

　読み上げ中は、次のコマンドを使用できます。
　・次の空白の行　　　　　　　　　　Space
　・段落の先頭　　　　　　　　　　　Ctrl + ←
　・次の記号段落（●や
