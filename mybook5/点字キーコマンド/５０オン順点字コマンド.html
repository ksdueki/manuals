５０オン順点字コマンドでは、キーのあとに 有効な画面を次の表記で表し、
続けて コマンド名を記述しています。
　[メニュー]は メニュー画面で有効なコマンド
　[一覧] は 一覧画面で有効なコマンド
　[本文] は 本文参照画面で有効なコマンド
　[メディア]は メディア再生画面で有効なコマンド
　[デイジー]は デイジー再生画面で有効なコマンド
　表記のない場合は、すべての画面で有効なコマンドであることを意味します。

【あの行】

あ + Space 選択項目の決定
い + Space [一覧/本文/メディア 既読 未読を切り替え]
お + Space [本文 カーソル位置から全文読み]
お + 無変換 [本文 文頭から全文読み]
【かの行】
か + Space [本文 カーソル位置 読み上げ]
【さの行】
さ + Space [メディア 現在の再生位置読み上げ]
し + Space [本文 しおりを付ける]
し + 無変換 一覧情報の読み上げ
す + Space [デイジー/メディア 再生スピード 上げる]
す + 無変換 [デイジー/メディア 再生スピード 下げる]
せ + Space [一覧/本文 本棚に保存]
せ + 無変換 [メニュー/一覧 先頭項目へ] [本文 段落の先頭]
[メディア 先頭から再生]
そ + Space [メニュー 本棚の並び替え]
【たの行】
つ + Space [本文 追加コピー]
つ + 無変換 [一覧 貼り付け]
て + Space [本文 点字文書の読み方切り替え]
と + Space [一覧 先頭の未読 図書へ] [本文 文頭]
【なの行】
な + Space [メニュー/一覧 名前の変更]
ね + Space [メニュー 5個単位で次の項目へ移動] [一覧 複数 下へスクロール]
[本文 つぎ 行タブ移動]
ね + 無変換 [一覧 次のヘッダへ]
【はの行】
は + Space [メニュー 5個単位で前の項目へ移動] [一覧 複数 上へスクロール]
[本文 まえ 行タブ移動]
は + 無変換 [一覧 前のヘッダへ]
ふ + Space [本文 範囲選択モード]
ほ + Space [メディア/デイジー 音量 上げる]
ほ + 無変換 [一覧/本文 コピー] [メディア/デイジー 音量 下げる]
【まの行】
ま + Space [本文 次のしおり]
ま + 無変換 [本文 前のしおり]
み + Space [一覧 次の未読 図書へ]
み + 無変換 [一覧 前の未読 図書へ]
む + 無変換 PC Talker　音声停止／再開
め + Space マイブックの終了
め + 無変換 [一覧 メニュー選択画面へ移動]
も + Space [一覧/本文 検索]
【やの行】
よ + Space 前回終了位置から読み上げ
【らの行】
ら + Space [一覧 最終の未読 図書へ] [本文 文末]
ら + 無変換 [メニュー/一覧 最終項目へ] [本文 段落の末尾]
る + Space [本文 段落読み]
る + 無変換 [本文 ルビの読み方切り替え]
【わの行】
わ + Space [本文 １行読み]
を + 無変換 [一覧/本文 すべて選択]

【点の組み合わせ】
5の点 + Space [一覧 削除]
6の点 + Space [一覧 メニュー選択・前のジャンルへ戻る]
[本文/メディア/デイジー 一覧に戻る]
2 5の点 + 無変換 [本文 前の行]
2 6の点 + 無変換 [本文 カーソルから段落末尾読み]
3 6の点 + 無変換 [本文 次の行]
4 5の点 + Space [一覧/本文 前を検索]
4 6の点 + Space [一覧/本文 次を検索]
2 3 5 6の点 + 無変換 [本文 段落先頭からカーソル手前読み]
3 4 5 6の点 + Space [メディア 指定時間から再生]

【点字周辺キー】
3の点 右下 X 点字符号読み
3の点 左下 Z JISコード読み
2の点 右下 C 識別読み
4の点 右下 M 上下カーソル移動行読み On/Off
1の点 右下 V 数字位取り読み　On/Off
3の点 上 W 日本語変換時の読み方切り替え
5の点 右下 , フォネティック読み　On/Off
1の点 右隣下 B 句点括弧読み　On/Off
6の点 右隣下 ? 英語読み　On/Off
