音声デイジーおよびマルチメディアデイジーの再生中に有効なコマンドです。

再生の終了　　　　　　　　　　　6の点 + Space
　　　　　　　　　　　　　　　　Escape または BackSpace
再生の一時停止と再開　　　　　　Space

次のフレーズ　　　　　　　　　　→
前のフレーズ　　　　　　　　　　←
次の見出し　　　　　　　　　　　↓
前の見出し　　　　　　　　　　　↑
次のページ　　　　　　　　　　　PageDown
前のページ　　　　　　　　　　　PageUp
ページジャンプ　　　　　　　　　Ctrl + J
次のしおり　　　　　　　　　　　M
前のしおり　　　　　　　　　　　Shift + M

ミュート　　　　　　　　　　　　テンキーの数字 0
音量を上げる　　　　　　　　　　ほ + Space　（ボリュームの ほ）
音量を下げる　　　　　　　　　　ほ + 無変換
再生スピードを上げる　　　　　　す + Space　（スピードの す）
再生スピードを下げる　　　　　　す + 無変換
再生スピードを標準に戻す　　　　Ctrl + テンキーの数字 0
右スピーカーのバランスを上げる Ctrl + Shift + →
左スピーカーのバランスを上げる Ctrl + Shift + ←
カーソル位置 読み上げ 　　　　　F9
