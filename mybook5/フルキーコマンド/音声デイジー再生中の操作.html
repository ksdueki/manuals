音声デイジーおよびマルチメディアデイジーの再生中に有効なコマンドです。

再生の終了　　　　　　　　　　　EscapeまたはBackSpace
再生の一時停止と再開　　　　　　Spaceまたはテンキーの数字 0

次のフレーズ　　　　　　　　　　→
前のフレーズ　　　　　　　　　　←
次の見出し　　　　　　　　　　　↓
前の見出し　　　　　　　　　　　↑
次のページ　　　　　　　　　　　PageDown
前のページ　　　　　　　　　　　PageUp
ページジャンプ　　　　　　　　　Ctrl + J
次のしおり　　　　　　　　　　　M
前のしおり　　　　　　　　　　　Shift + M

音量を上げる　　　　　　　　　　Ctrl + ↑
音量を下げる　　　　　　　　　　Ctrl + ↓
再生スピードを上げる　　　　　　Ctrl + →
再生スピードを下げる　　　　　　Ctrl + ←
再生スピードを標準に戻す　　　　Ctrl + テンキーの数字 0
右スピーカーのバランスを上げる Ctrl + Shift + →
左スピーカーのバランスを上げる Ctrl + Shift + ←
カーソル位置 読み上げ 　　　　　F9
