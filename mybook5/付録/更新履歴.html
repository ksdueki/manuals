Ver.5.10

2019年10月7日
 今回のバージョンアップでは以下の機能が追加されました。

・サピエ図書館の分類別検索にページ移動機能を追加
　分類別検索にページを切り替える機能が追加されました。
　　・Ctrl + J　　指定したページに移動する
　　・Alt + →　　次のページに素早く移動する
　　・Alt + ←　　前のページに素早く移動する

　分類別検索の簡易区分へ簡単に戻る事ができるようになりました。
　　・Escape　　　一つ上の区分へ戻る
　　　※← や BackSpace は、これまで通りひとつ前の画面に戻ります。

・デイジーを高音質で録音
　PC-Talker Neoの音声エンジンを使用して、より高音質なデイジー録音ができるようになりました。
　　※高音質な音声エンジンを使用してデイジー録音するには、PC-Talker Neoのインストールと有効なライセンスが必要です。

以下は機能や動作の改善についてです。

  • 一部のEPUBファイルがまれに開けなくなる動作を改善しました。
  • 拡大率を変更した場合にカーソル位置が変更されてしまう現象を改善しました。
  • サピエ図書館において、シネマデイジーの未読・既読が保存されない動作を改善しました。
  • サピエ図書館での「最近開いた履歴」で「誘導検索」した結果のデイジーを正常に保存できない動作を改善しました。
  • 図書タイトルの点字ディスプレイ表示を改善しました。
    ※タイトルを点字ディスプレイ表示した後に、項目の概要を点字ディスプレイ表示したためタイトル表示を阻害していた動作を改善。
  • 初めて開いた青空文庫の図書をデイジー録音する場合に、初期タイトルが設定されない問題を改善しました。
  • デイジー録音機能を使用して作成したデイジーが、特定の状況で前の続きからがおかしくなる問題を改善しました。
  • 一部のWindows10環境で、起動時にZIPファイルに関するセットアップが不完全と表示されてしまう問題を改善しました。
  • 一部の環境で .Net FrameWorkが必要であるという誤ったメッセージが表示される動作を改善しました。
  • Web版を使用しており .Net FrameWorkがインストールされていない環境でのインストール動作を改善しました。
    ※.Net FrameWorkのインストールを行うにはインターネット接続が必要です。
  • 読み上げ終了位置のクラウド保存の対象をマニュアルに注記しました。
    ※対象は デイジー、テキストデイジー、EPUB であり、なおかつ図書を識別する情報が設定されている図書となります。
