目次の中には、旅行に関する録画番組や、民謡を録音したものなど、メディアを楽しむことができるものがあります。
視聴したいタイトルを選択してEnterを押すと、そのメディアの再生が始まります。
このとき、メディアのデータファイルを放送局のホームページから自動的に読み込むため、実際に再生が始まるまで時間がかかることがあります。

メディアの再生中、一時停止中に次のコマンドを使用できます。
　・再生の一時停止と再開            ↓
　・再生の早送り                    →
　・再生の巻き戻し                  ←
　・先頭まで巻き戻して再生          Home
　・指定時間からの再生              Ctrl + J
    ※点字コマンドでは              3 4 5 6の点 + Space
    ※再生を開始する時間【分】【秒】を数値で入力し、Enterを押します。
      入力した時間の位置から再生が始まります。
　・ミュート                        テンキーの数字 0
    ※ミュートとは、音を消したまま再生させることです
　・音量を上げる                    Ctrl + ↑
　・音量を下げる                    Ctrl + ↓
    ※この音量は、再生音の大きさであり、PC-Talkerの音量には影響しません
　・左スピーカーのバランスを上げる  Ctrl + ←
　・右スピーカーのバランスを上げる  Ctrl + →
    ※これは、ステレオスピーカーの音量バランスを調整する機能です
　・画像の大きさ切り替え                テンキーの数字 1
    ※画像をウィンドウサイズに合わせるか、通常サイズにするかを切り替えます
　・メディア情報読み上げ            Alt + Ctrl + G
    ※次のメディア情報を読み上げます。
      1.メディアタイトル
      2.メディアタイプ  『ビデオ』『オーディオ』など
      3.メディア全体の再生時間
      4.メディアの著作権
      5.現在の状態      『再生中』か『一時停止』
　・現在の再生位置の読み上げ        Alt + Ctrl + F9
　　※現在の再生位置までの時間と、全体の再生時間を読み上げます

再生を中止して、トピック一覧に戻りたいときは、Escapeを押します。

※再生中にコマンドが効かないことがあります。
再生中、パソコンのCPUは再生処理にほとんどの能力を使います。
再生するデータ量が多く、パソコンのCPUに余力が無い状態では、コマンドの反応が遅れたり、コマンドが効かなかったりすることがあります。

※メディアが再生できないことがあります。
この場合、メディアを再生できない旨のメッセージが出ます。
また、その原因として、次のことが考えられます。
　1. 放送局のホームページが、何かしらの事情で閉じられている。
     《対応》しばらく時間をおいてください。
　2. 中継番組の場合、その時間は放送中でない。
     《対応》放送中のときに再生してください。
　3. リンク先のデータファイルがホームページ上から削除されている。
     《対応》これに関しては諦めるしかありません。
　4. ファイアウォールやルータによって、メディア用の受信ポートが閉じられている。
     職場や公共施設では、セキュリティ管理上、閉じられていることがあります。
     《対応》管理者に相談してください。
