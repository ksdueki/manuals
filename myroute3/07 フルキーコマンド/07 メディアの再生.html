◆本文自体がメディアになっているときのコマンド

  1. トピック一覧に戻る Escape または BackSpace
  2. このトピックの開封 未開封を切り替え Insert
  3. 次のトピックへ切り替え Alt + ↓
  4. 前のトピックへ切り替え Alt + ↑
  5. 巻き戻し ←
  6. 早送り →
  7. 一時停止と再開 ↓ または Enter
  8. 先頭位置から再生 Home
  9. 指定時間から再生 Ctrl + J
  10. 音量 上げる Ctrl + ↑
  11. +
  12. 音量 下げる Ctrl + ↓
  13. -
  14. 左スピーカーのバランスを上げる Ctrl + ←
  15. 右スピーカーのバランスを上げる Ctrl + →
  16. ミュート切り替え テンキー 0
  17. 画像の大きさ切り替え テンキー 1
  18. メディア情報読み上げ Alt + Ctrl + G
  19. 現在の再生位置読み上げ Alt + Ctrl + F9

※注意

メディア再生中、パソコンのCPUは、再生処理にほとんどの能力を使います。

再生するデータ量が多く、パソコンのCPUに余力が無い状態では、コマンドの反応が遅れたり、コマンドが効かなかったりすることがあります。

もともとの大きさが小さい動画を、拡大したりフルスクリーン表示にした場合も、同様の理由でコマンドの反応が遅れたり、コマンドが効かないことがあります。

また データによっては大音量で再生されるものがあります。

ヘッドフォンなどを使用されている場合、危険ですのでご注意ください。
