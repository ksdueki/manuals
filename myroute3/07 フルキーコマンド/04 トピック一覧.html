1. チャンネル選択・目次の操作へ戻る ← または Escape または BackSpace
  2. 上のトピックへ ↑
  3. 下のトピックへ ↓
  4. 先頭トピックへ Home
  5. 最終トピックへ End
  6. 複数 上へスクロール PageUp
  7. 複数 下へスクロール PageDown
  8. 先頭の未読トピックへ Ctrl + Home
  9. 最終の未読トピックへ Ctrl + End
  10. 次の未読トピックへ Ctrl + Shift + ↓
  11. 前の未読トピックへ Ctrl + Shift + ↑
  12. 前のヘッダへ Ctrl + PageUp
  13. 次のヘッダへ Ctrl + PageDown
  14. 地名、駅名の切り替え F4 ※タウンページ検索中の場合のみ有効
  15. 本文の読み上げ → ※読み上げ停止は ←
  16. 本文の読み上げ Space ※読み上げ中のカーソル移動が可能
  17. 連続トピック読み Alt + F10
  18. 選択中のトピック題名を再読み Shift
  19. メディアの再生 → ※再生中の操作はできません
  20. メディアの再生音量 上げる +
  21. メディアの再生音量 下げる -
  22. 次のチャンネルへ切り替え Alt + ↓ ※チャンネル選択の項目が移動します
  23. 前のチャンネルへ切り替え Alt + ↑ ※チャンネル選択の項目が移動します
  24. 開封 未開封を切り替え Insert
  25. 開封済みにする Ctrl + Q
  26. 未開封に戻す Ctrl + I
  27. すべて開封する Ctrl + Shift + Q
  28. 保存ボックスへ保存 Ctrl + S
  29. 名前を付けて保存 Ctrl + Shift + S / Ctrl + Shift + F2
  30. コピー Ctrl + C / Ctrl + Insert
  31. 貼り付け Ctrl + V / Shift + Insert
  32. 削除 Delete
  33. すべて選択 Ctrl + A
  34. 検索 Ctrl + F
  35. 次を検索 F3
  36. 前を検索 Shift + F3
  37. 本文の印刷 Ctrl + P
  38. 本文の参照へ Enter
  39. メディアの再生へ Enter
  40. トピック参照画面へ移動 Tab
  41. チャンネル選択画面へ移動 Shift + Tab
