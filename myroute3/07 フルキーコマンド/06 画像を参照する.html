本文が画像データの場合は、『画像』と音声ガイドされ画像参照画面が表示されます。
画像参照画面では、以下の操作を実行できます。

画像の印刷                         Ctrl + P

◆表示中の画像のサイズを変更するには以下のキーを使用します。
拡大                               PageUp
縮小                            　 PageDown
原寸大                             Ctrl + Home
  ※画像を実際の大きさで表示します。
色の反転                           Insert
コピー                             Ctrl + C
                                   Ctrl + Insert
画像参照画面の終了                 Escape または BackSpace
