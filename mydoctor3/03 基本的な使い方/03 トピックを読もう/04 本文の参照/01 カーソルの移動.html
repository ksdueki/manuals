◆カーソル移動の基本は、← →で左右の文字へ、↑↓で上下の行へ移動です。
　他にも、次のコマンドで、自由に移動することができます。
　・次の文節へ移動 Ctrl + →
　・前の文節へ移動 Ctrl + ←
　行の先頭が行頭で、最後の文字が行末です。
　・行頭へ移動 Home
　・行末へ移動 End
　文章の先頭から、改行文字で行が改められるまでのブロックが段落です。
　・前の段落へ移動 Ctrl + ↑
　・次の段落へ移動 Ctrl + ↓
　文章の先頭が文頭で、最後が文末です。
　・文頭へ移動 Ctrl + Home
　・文末へ移動 Ctrl + End

◆現在のカーソル位置を音声で知るには、F9を押します。
　行 列の順で『5の1』のように読み上げます。

◆カーソル移動時の文字の読み方を変更するには、F8を押します。
　カーソル移動したときの文字の読み方には、
　簡易読み 簡単な 易しい説明の読みです。
　音訓読み 六点漢字の 音読みと訓読み で表現した読みです。
　音読み 音読みです。
　詳細音訓読み 簡易読みよりも詳しい読みです。
　詳細区分読み 詳細音訓読みに､漢字の区分を加えた読みです。
　識別読み 全角 半角や、文字の種類も加えた詳しい読みです。
　ビープ 簡単な電子音を出力します。
　と、７つの読み方があります。
　F8で、これらの読み方を順番に切り替えることができます。

◆カーソル移動の点字コマンドは、次の通りです。
　・↑ 2 5の点 + Space
　・↓ 3 6の点 + Space
　・→ 2の点 + Space
　・行頭に移動 せ + 無変換（ 行の先頭の せ ）
　・行末に移動 ら + 無変換（ 行のラストの ら ）
　・前の段落に移動 2 5の点 + 無変換
　・次の段落に移動 3 6の点 + 無変換
　・文頭に移動 と + Space（ 文章のトップの と ）
　・文末に移動 ら + Space（ 文章のラストの ら ）
　・カーソル位置の読み上げ か + Space（ カーソル位置の か ）
