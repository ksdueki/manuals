本文の読み上げ操作には、読み上げ確認したい範囲に応じた、全文読みや段落読み、１行読みなどがあります。
また本文を読み上げている際に、本文中のリンク部分や画像部分が聞き分けられるよう、その部分は音程が変化して読み上げます。
通常の音程より高く読み上げる場合、その部分はリンクであることを表します。
通常の音程より低く読み上げる場合、その部分は画像の代わりに用いられるテキストです。

※補足
本文の中には、別の本文や画像、動画を参照するための情報が埋め込まれた部分があります。
この部分を「リンク」していると言います。
そして、このリンクを表す部分にカーソルを移動させて、そこからリンク先の内容を参照することを「リンクを実行する」と言います。
リンクの実行については《3-3-4-4. リンクを実行する》で、表示画像の確認については《3-3-4-6. 表示画像を確認する》でそれぞれ説明していますので、参照してください。

◆全文読み
　Alt + F10で 現在のカーソル位置から読み上げを開始します。
　読み上げ中は、カーソルが、読み上げている場所を追いかけながら進んでいきます。
　また、カーソルを操作して、読み上げ位置を自由に移動することもできます。
　読み上げを中断したいときは、Escapeを押します。

◆現在のカーソル上の文字や文章を読み上げるコマンドも豊富にあります。
　・カーソル行の読み上げ Alt + F8
　・行頭からカーソル位置の手前までの読み上げ
PC-Talker Ctrl + Alt + H
VDMW Ctrl + Alt + J
　・カーソル位置から行末までの読み上げ
PC-Talker Ctrl + Alt + K
VDMW Ctrl + Alt + L
　・段落読み Alt + F9
　・カーソル文字の再読み
PC-Talker Ctrl + Alt + R
VDMW Ctrl + Alt + G
　・カーソル文字の点字符号読み
PC-Talker Ctrl + Alt + F
VDMW Ctrl + Alt + X
　・カーソル文字のJISコード読み
PC-Talker Ctrl + Alt + I
VDMW Ctrl + Alt + N

◆本文読み上げの点字コマンドは、次の通りです。
　・全文読み お + Space （オールの お）
　・１行読み わ + Space （ワンラインの わ）
　・段落読み る + Space （以前は、ループ読みと呼んでいた）
　・行頭からカーソル手前読み れ下がり + 無変換 ※PC-Talkerのみ
　・カーソルから行末読み ら下がり + 無変換 ※PC-Talkerのみ

　以下は、六点キー周辺に配置されているキーで実行します。
　・カーソル文字の再読み 1の点 右隣 G
　・カーソル文字の点字符号読み 3の点 右下 X
　・カーソル文字のJISコード読み 3の点 左下 Z
