一覧の操作も、目次と同じように簡単なキー操作で行うことができます。
　・上のトピックを選択          ↑
　・下のトピックを選択          ↓
　・選択しているトピックの本文読み上げ  → または Space
　・目次(ジャンル)へ戻る                ← または Escape

目次の操作と合わせても、上下左右の矢印キーだけを覚えておくと、トピックを読むまでの操作はできますね。
この他、一覧の操作に共通して、次のキー操作ができます。
　・先頭のトピックへ移動        Home
　・最終のトピックへ移動        End
　・上へスクロール移動          PageUp
　・下へスクロール移動          PageDown
　・キーボード右側に並んでいるテンキーを押して、入力番号のトピックへジャンプ
    例えば［5］を押すと５番目のトピックへジャンプします。
    ５８番目のトピックへジャンプしたいときは、［5］［8］と続けて押します。
    ※テンキーのナムロックがオフ状態のときはジャンプできません。
      ナムロック状態がオンであることを確認してお使いください。

◆一部のジャンルのトピック一覧でのみ、更に効率よくトピックを読む工夫がされています。
新聞の場合、トピックの公開日が分かるように、今日、昨日、今週、先週、古いの単位で公開日を区切る項目がトピックの中に混じってあります。
この区切り項目を「ヘッダ」と呼びます。
例えば『7月14日 水曜日 昨日のトピック』という項目がヘッダです。
そして、このヘッダの下にその公開日のトピックが並んでいます。
各ヘッダへは、次のジャンプコマンドで素早く移動できます。
　・前のヘッダ                  Ctrl + PageUp
　・次のヘッダ                  Ctrl + PageDown

◆トピックに、開封・未開封の状態を設定することができます。
選択しているトピックの本文は→ や Spaceで読み上げたとき自動的に開封状態となりますが、次のコマンドで開封・未開封の状態を変更できます。
開封したトピックを選択したときは、『開封』と音声ガイドされます。
　・開封・未開封の切り替え          Insert
　・開封状態にする              Ctrl + Q
    ※点字コマンドでは          ひ + Space（開くの ひ）
　・未開封状態に戻す            Ctrl + I
    ※点字コマンドでは          し + Space（閉めるの し）
開封していないトピックを選択したいときは、次のコマンドでジャンプできます。
　・先頭の未開封トピック        Ctrl + Home
　・最終の未開封トピック        Ctrl + End
　・前の未開封トピック          Ctrl + Shift + ↑
　・次の未開封トピック          Ctrl + Shift + ↓

◆一覧情報読み上げ
トピック一覧で Alt + F1を押して、一覧情報読み上げを実行した場合『目次 ニュース 読売 医療ニュースのトピック』のように、現在一覧しているジャンルの位置を読み上げます。
そして『100件中 8件目を選択しています』と、現在の状態も一緒に読み上げます。

◆選択したトピックの本文を、カーソルを使ってじっくりと読み上げ確認したいときは、Enterを押して、トピックの参照画面へ移動します。
トピックは、参照したとき自動的に開封状態になります。
※《3-3-4. 本文の参照》を参照。

◆選択したラジオ・テレビの番組を再生したいときは、Enterを押してメディアの再生へ移動します。
※《3-3-5. メディアの再生》を参照。
