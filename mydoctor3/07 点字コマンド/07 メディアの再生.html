メディアの再生には本文自体がメディアになっているときと、テキスト形式の本文中にメディアへのリンクが埋め込まれているときの２種類があります。

◆本文自体がメディアになっているときのコマンド

トピック一覧に戻る                 6の点 + Space または Escape
このトピックの開封 未開封を切り替え    い + Space
次のトピックへ切り替え                 Alt + ↓
前のトピックへ切り替え                 Alt + ↑

早送り                             →
巻き戻し                           ←
一時停止と再開                     あ + Space または ↓
先頭位置から再生                       せ + 無変換
指定時間から再生                   3 4 5 6の点 + Space
音量 上げる                        Ctrl + ↑
音量 下げる                        Ctrl + ↓
左スピーカーのバランスを上げる     Ctrl + ←
右スピーカーのバランスを上げる     Ctrl + →
ミュート切り替え                       テンキー 0
画像の大きさ切り替え                   テンキー 1

メディア情報読み上げ               Alt + Ctrl + G
現在の再生位置読み上げ             Alt + Ctrl + F9


◆本文内のリンク先に埋め込まれているメディアを参照するときのコマンド

本文が動画データや一部の音声データの場合は、『メディア』と音声ガイドされてメディア参照画面が表示されます。
メディア参照画面が表示されたあとは、メディアが読み込まれ自動的に再生が開始されます。
なんらかの理由で再生できない場合は『このメディアはただいま 使用できません』と音声ガイドされます。

メディアの再生が終了すると『再生 終わり』と音声ガイドされます。

メディア参照画面では、以下の操作を実行できます。
一時停止と再開                     あ + Space または ↓
早送り                             →
巻き戻し                           ←
次へスキップ                       Alt + →
前へスキップ                       Alt + ←
　※スキップ操作は、スキップ区切りの付いたメディアのみで有効です。
先頭位置から再生                   せ + 無変換
音量を上げる                       Ctrl + ↑
                                   +
音量を下げる                       Ctrl + ↓
                                   -
  ※この音量は再生音の大きさであり、PC-Talkerの音量には影響しません
ミュート切り替え                       テンキー 0
　※ミュートとは、音を消したまま再生させることです
拡大                               は + Space
縮小                               ね + Space
原寸大                             と + Space
　※メディアが動画の場合、動画の実際の大きさで表示します。
フルスクリーン                     F11
　※再生画面を、モニターいっぱいに拡大します。
メディア参照画面の終了             6の点 + Space
                                   4の点 + Space
　※『メディア オワリ』と音声ガイドされ、メディア参照画面を終了します。

※注意
メディア再生中、パソコンのCPUは、再生処理にほとんどの能力を使います。
再生するデータ量が多く、パソコンのCPUに余力がない状態では、コマンドの反応が遅れたり、コマンドが効かなかったりすることがあります。
もともとの大きさが小さい動画を、拡大したりフルスクリーン表示にした場合も、同様の理由でコマンドの反応が遅れたり、コマンドが効かないことがあります。
また データによっては大音量で再生されるものがあります。
ヘッドフォンなどを使用されている場合、危険ですのでご注意ください。
