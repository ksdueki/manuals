■基本操作
　左の文字へ移動         ←
　右の文字へ移動         →
　上の行へ移動           ↑
　下の行へ移動           ↓

　行の先頭が行頭で、最後の文字が行末です。
　行頭へ移動             Home
　行末へ移動             End

　文章の先頭から、改行文字で行が改められるまでのブロックが段落です。
　前の段落へ移動         Ctrl + ↑
　次の段落へ移動         Ctrl + ↓

　文書の先頭が文頭で、最後が文末です。
　文頭へ移動             Ctrl + Home
　文末へ移動             Ctrl + End
　音声では『トップポジション』『ラストポジション』とガイドします

■表内の移動
　表を使っているページは、認識文書にも その表イメージが取り込まれます。
　表は、縦横の並びを マス目状に罫線で区切って表し、縦の並びを列、横の並びを行と呼びます。罫線で区切られた１つ１つのマス目を セルと呼び、このセルの中に文章が書き込まれています。
　↑ ↓ ← → を押して、表内をなぞるようにカーソル移動します。
　この場合、罫線の上にもカーソルは移動します。

　次のセル、前のセルというように、セルからセルへ移動できます。
　次のセルへ移動         Ctrl + →
　前のセルへ移動         Ctrl + ←

■複数ページの文書移動
　複数のページを連続読み取りしたときや、認識文書を追加しながら読み取りしたとき
　各ページの先頭位置へ移動できます。
　前のページへ移動         PageUp
　次のページへ移動         PageDown
