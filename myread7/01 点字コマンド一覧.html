KTOSの漢点字入力を使用しているかたは、「点字符号+無変換」の部分を「点字符号+変換」に置き換えて使用してください。


●全般
  マイリードの終了                 め + Space
  マニュアルを読む                 め + 無変換
  プルダウンメニューの操作に移る   6の点 + 無変換
  Enterの代替                      あ + Space
  Escapeの代替                     6の点 + Space


●拡大読書画面の操作
  画像の右90度回転                 か + 無変換
  画像の色 反転して表示            い + 無変換
  拡大読書画面の終了               6の点 + Space または め + Space
  》以下は、認識画像の拡大読書中で有効
      先頭領域へ 表示位置を移動    と + Space
      最終領域へ 表示位置を移動    ら + Space


●取り込み
  画像を開く                       ろ + Space
  PDFを開く                        ろ + 無変換


●データの保存と印刷
  認識文書の保存                   せ + Space
  認識文書の印刷                   ふ + 無変換
  画像の保存                       ひ + Space
  画像の印刷                       ふ + Space
  PDFへ保存                        ひ + 無変換

●認識
  ページ削除                       れ + Space
  すべて削除                       く + Space

  行頭へ                           せ + 無変換

  次の段落へ                       3 6の点 + 無変換
  文頭へ                           と + Space

●【認識文書】…… カーソル文字の読み上げ
  カーソル位置の読み上げ           か + Space       ※『原稿番号 行 列』

●【認識文書】…… なめらか読みの読み方切り替え
  句点・括弧の読み方切り替え           1の点の右下右
  数字位取り読みオンオフ           1の点の右下
  英語読み切り替え                     6の点の右下右
●【認識文書】…… 文書の読み上げ
  １行読み                         わ + Space
  行頭からカーソル手前読み         れ下がり + 無変換
  カーソルから行末読み             ら下がり + 無変換
  段落読み                         る + Space
  表セル内読み                     わ + 無変換
  文頭から全文読み                 お + 無変換
  カーソルから全文読み             お + Space
  読み上げの中断                   6の点 + Space
●【認識文書】…… 日本語変換中の読み上げ
  反転表示内の読み上げ             れ下がり + 無変換
  反転表示以降の読み上げ           ら下がり + 無変換

●【認識文書】…… その他の読み上げ
  入力状態の読み上げ               に + 無変換
  ページ設定の読み上げ             へ + 無変換

●【認識文書】…… 文字の挿入・削除
  挿入・上書きの切り替え                 い + Space

  カーソル上の文字削除             5の点 + Space
  カーソル手前の文字削除           4の点 + Space
●【認識文書】…… 編集
  元に戻す                         6の点の右上

  切り取り                         ぬ + Space

  貼り付け                         つ + 無変換
  検索                             も + Space
  次を検索                         4 6の点 + Space

  置換                             ち + Space
