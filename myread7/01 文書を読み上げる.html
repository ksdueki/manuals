■全文読み
　全文読みのコマンドには、次の２つがあります。
　Alt + F5で  文頭から読み上げを開始します。
　Alt + F10で 現在のカーソル位置から読み上げを開始します。
読み上げ中は、カーソルが、読み上げている場所を追いかけながら進んでいきます。
文書の最後まで読み上げたら『おわり』と音声ガイドして読み上げを終了します。
読み上げを中断したいときは、Escapeを押します。『パス』と音声ガイドし、読み上げを終了します。このとき、カーソル位置は、最後に読み上げていた文章の近くまで進んできています。ですから、ここだと思った内容が読み上げられたら、Escapeで読み上げを停止して、その場所までカーソル移動すると目的の場所に早くたどり着けます。

■全文読み上げ中の操作
　全文読みの最中に、内容を聞き逃してしまったり、もう一度 注意深く聞いたりしてみたいときは、読み上げ位置を戻すことができます。逆に、今聞いている内容をスキップさせたいときは、読み上げ位置を進めることができます。
　以下に、全文読み上げ中の操作コマンドを紹介します。

　段落単位で読み上げ位置を移動します。
　段落の先頭に戻る       ←
　次の段落先頭に進む     ↓
　前の段落先頭に戻る     ↑

　空白や句読点で区切られた文節へ、読み上げ位置を進めます。
　次の文節に進む         →

　行単位で読み上げ位置を移動します。
　次の行へ進む           Enter
　前の行へ戻る           BackSpace

　ページ単位で読み上げ位置を移動します。
　ページの先頭へ戻る     Home
　次のページへ進む       PageDown
　前のページへ進む       PageUp

　また、全文読み上げ中に 音声スピード・音声ボリュームを切り替えできます。
　音声スピード切り替え       F7
　音声ボリューム切り替え     F11

■１行読み
　カーソル行の読み上げ   Alt + F8

　行頭からカーソル位置の手前までの読み上げ
　　PC-Talker            Ctrl + Alt + H
　　VDMW                 Ctrl + Alt + J

　カーソル位置から行末までの読み上げ
　　PC-Talker            Ctrl + Alt + K
　　VDMW                 Ctrl + Alt + L

■段落読み
　Alt + F9で カーソルのある段落の先頭から、その段落の最後までを読み上げます。

■表の読み上げ
　全文読み上げ中は、表をセル単位で 順番に読み上げていきます。

　セル内の文章を始めから終わりまで読み上げ確認する便利なコマンドがあります。
　表のセル内読み         Alt + F7

■カーソル上の文字情報を読み上げる、いろんなコマンドを紹介します。
　カーソル文字の再読み
　カーソル上の文字を再度読み上げます。
　　PC-Talker            Ctrl + Alt + R
　　VDMW                 Ctrl + Alt + G

　カーソル文字の点字符号読み
　KTOSご使用の場合、六点漢字の符号で読み上げます。
　KTOS8ご使用の場合、漢点字の符号で読み上げます。
　　PC-Talker            Ctrl + Alt + F
　　VDMW                 Ctrl + Alt + X

　カーソル文字のJISコード読み
　シフトJISコードに続けて、JISコードの順番で読み上げます。
　　PC-Talker            Ctrl + Alt + I
　　VDMW                 Ctrl + Alt + N
